### Docer volumes on different OS

```
# If Windows CMD
> docker run -v %cd%:/app app:1
# If Windows PowerShell
> docker run -v ${PWD}:/app app:1
# If Windows Git Bash
> docker run -v /$(pwd):/app app:1
```
